# Task: Create Offer

**Command:** `*create-offer`
**Agent:** Hormozi Chief + Hormozi Offers + Hormozi Pricing
**Estimated Time:** 50 minutes

## Description
Cria uma oferta irresistível usando o framework completo do $100M Offers de Alex Hormozi. Transforma um produto/serviço genérico em uma Grand Slam Offer que faz o prospect sentir que seria estúpido dizer não. Inclui value stack, pricing strategy, garantia e naming.

## Inputs
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| product | string | yes | Produto ou serviço atual |
| target_audience | string | yes | Nicho, dores, desejos e sonhos do cliente ideal |
| current_price | string | no | Preço atual (se houver) |
| delivery_method | string | yes | Como é entregue (digital, presencial, híbrido, SaaS) |
| dream_outcome | string | yes | O resultado dos sonhos que o cliente quer alcançar |
| competitors | string | no | Principais alternativas no mercado |

## Steps
1. Hormozi Chief avalia o briefing e identifica oportunidades de diferenciação
2. Hormozi Offers aplica o Value Equation: Dream Outcome × Perceived Likelihood / Time Delay × Effort & Sacrifice
3. Constrói o Value Stack com deliverables, bonuses, e componentes que aumentam valor percebido
4. Hormozi Pricing define preço baseado em valor (não em custo), aplica estratégia de ancoragem
5. Define garantia (unconditional, conditional, ou anti-guarantee) e naming magnético para a oferta
6. Hormozi Chief consolida em Grand Slam Offer completa

## Output
Documento da Grand Slam Offer contendo: Value Equation preenchida, Value Stack detalhado (core offer + 3-5 bonuses com valor atribuído), pricing com justificativa, tipo de garantia com wording, nome da oferta, e comparativo "preço vs. valor" para uso em vendas.

## Example
**Input:** Produto: consultoria de marketing. Audience: donos de e-commerce com faturamento de R$50k-500k/mês. Dream outcome: dobrar faturamento em 6 meses.
**Output:** Grand Slam Offer "Programa Acelerador E-commerce 2X": Core (consultoria semanal 1:1 — R$12.000), Bonus 1 (auditoria completa de funil — R$5.000), Bonus 2 (templates de email — R$3.000), Bonus 3 (acesso comunidade — R$4.000), Bonus 4 (sessão de emergência — R$2.000). Valor total: R$26.000. Preço: R$8.000. Garantia condicional: "Se não aumentar pelo menos 30% em 90 dias com implementação verificada, devolvemos tudo."
