## Core Principles
1. **"So What?" Test** — Todo dado deve sobreviver ao teste: "E daí? O que eu faço com isso?" Se não gera ação, não merece estar no relatório.
2. **Macro + Micro conversions** — Focar apenas na venda final é cegueira. As micro-conversões (email signup, video view, PDF download) são os sinais vitais do funil.
3. **Segmentação é tudo** — Agregar dados é esconder a verdade. Segmente por fonte, dispositivo, localização, comportamento, e de repente os insights aparecem.
4. **Qualitativo + Quantitativo** — Analytics diz o "quê" e o "quanto". Para o "porquê", você precisa de surveys, usability testing e session recordings.
5. **10/90 Rule** — Invista 10% do budget em ferramentas e 90% em pessoas inteligentes para analisar. Ferramenta sem analista é dashboard bonito sem valor.

## Frameworks
### Digital Marketing & Measurement Model (DMMM)
Framework de 5 etapas: (1) Defina objetivos de negócio, (2) Defina goals para cada objetivo, (3) Defina KPIs para cada goal, (4) Defina targets para cada KPI, (5) Identifique segmentos de valor. Elimina o problema #1 de analytics: medir tudo sem saber por quê.

### See-Think-Do-Care Framework
Modelo de audiência baseado em intenção: See (maior audiência qualificada), Think (audiência com intenção comercial), Do (audiência com intenção de compra), Care (clientes existentes). Cada estágio tem conteúdo, KPIs e canais específicos.

### Trinity Strategy
Abordagem integrada: Clickstream Analysis (o quê), Multiple Outcomes Analysis (quanto) e Experimentation/Testing (por quê funciona). A combinação dos três produz insights que nenhum sozinho consegue.
