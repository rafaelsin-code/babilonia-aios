# Task: Positioning

**Command:** `*positioning`
**Agent:** Brand Chief + Al Ries + Byron Sharp
**Estimated Time:** 40 minutes

## Description
Define ou redefine o posicionamento estratégico da marca usando o framework de Al Ries (dono da categoria/posição mental) combinado com os princípios de distinção de Byron Sharp. Entrega um posicionamento claro, defensável e implementável.

## Inputs
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| brand_name | string | yes | Nome da marca |
| category | string | yes | Categoria atual de mercado |
| target_audience | string | yes | Público-alvo principal |
| competitors | string | yes | Concorrentes e seus posicionamentos |
| differentiators | string | yes | O que a marca faz diferente ou melhor |
| constraints | string | no | Restrições (budget, legacy, regulação) |

## Steps
1. Al Ries analisa o landscape de categorias e identifica posições mentais disponíveis
2. Byron Sharp avalia distinctive assets e oportunidades de mental/physical availability
3. Brand Chief cruza as análises e formula opções de posicionamento
4. Teste de posicionamento: cada opção é avaliada contra 5 critérios (relevante, diferenciado, crível, sustentável, simples)
5. Entrega posicionamento final com implementação

## Output
Documento de posicionamento contendo: mapa de categorias e posições ocupadas, análise de posições disponíveis, 3 opções de posicionamento com prós/contras, posicionamento recomendado, positioning statement formal, distinctive brand assets recomendados, e guia de implementação por canal.

## Example
**Input:** Marca: plataforma de cursos. Categoria: edtech. Concorrentes: Hotmart, Udemy, Coursera.
**Output:** Posição disponível: "a plataforma para quem quer ensinar, não para quem quer vender" (anti-infoproduto). Statement: "Para educadores independentes que valorizam profundidade, [marca] é a plataforma de cursos que prioriza aprendizado real sobre métricas de venda, porque acreditamos que educação de verdade é o melhor marketing."
