# Task: Community Plan

**Command:** `*community-plan`
**Agent:** Movement Chief + Community Weaver + David Spinks (data-squad)
**Estimated Time:** 40 minutes

## Description
Cria um plano completo de comunidade — desde estratégia e formato até operações e métricas. Define por que a comunidade existe, para quem, qual valor gera para membros e para o negócio, e como escalar sem perder qualidade.

## Inputs
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| business_goal | string | yes | Objetivo de negócio que a comunidade deve servir |
| target_members | string | yes | Perfil do membro ideal |
| community_size_target | string | yes | Tamanho-alvo em 6-12 meses |
| platform_preference | string | no | Plataforma preferida (Slack, Discord, Circle, etc.) |
| budget | string | yes | Budget disponível para community ops |
| existing_audience | string | no | Audiência existente que pode ser convertida |

## Steps
1. Define SPACES Model: qual valor primário a comunidade gera (Support, Product, Acquisition, Contribution, Engagement, Success)?
2. Community Weaver projeta Interaction Design e Community Rhythm
3. Define governance: regras, moderação, leadership, escalation
4. Cria launch plan: founding members, soft launch, public launch
5. Define métricas: connection density, belonging score, business impact

## Output
Plano de comunidade contendo: estratégia (SPACES value, target member persona), plataforma recomendada com justificativa, community rhythm (daily/weekly/monthly/quarterly), governance framework, launch plan em 3 fases (founding → soft → public), content & programming calendar, team & roles necessários, métricas e dashboards, e budget breakdown.

## Example
**Input:** Goal: reduzir churn de SaaS em 20%. Members: product managers que usam o SaaS. Target: 1.000 membros em 12 meses. Budget: R$5k/mês.
**Output:** SPACES: Success (peer-to-peer learning) + Product (feedback loop). Plataforma: Circle (melhor para professional communities). Rhythm: daily async posts, weekly "PM Office Hours" (live 45min), monthly "Show & Tell" (members apresentam uso criativo), quarterly "PM Summit" (evento especial). Launch: 50 founding members hand-picked → 200 soft launch com invite-only → public após 3 meses com waitlist. Métricas: 30-day retention >60%, connection density >3 connections/member, churn rate de membros ativos vs. não-membros.
